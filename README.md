## iOder
-An online ordering system
-course project of KTH ID2216

## React + TypeScript + Vite + Ant Design + Zustand

- use a React mobile template from https://github.com/xmllein/react-mobile

##  Project tech stack

- [x] react-router-dom 6.x
- [x] antd-mobile 5.x
- [x] zustand 4.x
- [x] axios
- [x] sass
- [x] postcss-pxtorem
- [x] vite-plugin-mock
- [x] mockjs
- [x] husky
